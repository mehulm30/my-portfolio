# 🚀 Mehul More — Personal Portfolio (React.js)

A modern, responsive, and animated developer portfolio built using **React.js**, transformed from a static HTML/CSS/JS website.  
This version includes background animations, AOS scroll effects, theme toggling, upgraded screen loader, reusable components, and clean folder architecture.

---

## 🎯 **Features**

### ✅ Modern UI & Smooth Animations  
- Glassmorphism sections  
- Floating orbit background + shooting-star effects  
- Smooth AOS scroll animations  

### ✅ Fully Responsive  
Works perfectly on phones, tablets, and desktops.

### ✅ Theme Support  
- Light & Dark theme toggle  
- Persistent theme stored in `localStorage`

### ✅ Upgraded Loader  
- Rotating sarcastic quotes  
- Smooth progress bar  
- Fades out into the website  
- Theme-aware design

### ✅ Component-Based Architecture  
Every section of the portfolio is modular and easy to update.

### ✅ Fast & Optimized  
Built with React best practices and minimal dependencies.

## ⚙️ Setup & Installation

### 1. Clone the repository

```bash
git clone https://github.com/mehulm30/my-portfolio.git
```

### 2. Navigate to project directory

```bash
cd my-portfolio
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run locally

```bash
npm start
```

---

## 📦 Build for Production

```bash
npm run build
```
